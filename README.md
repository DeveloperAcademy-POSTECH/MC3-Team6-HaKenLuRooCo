# :iphone: Today, Anbu

MC3-Team6-HaKenLuRooCo
팀 하켄루루코의 "오늘, 안부" 입니다.
"오늘, 안부"는 갱년기를 겪고 계신 부모님들에게 연락을 쉽게 할수 있도록 도와주는 앱입니다.

## :pushpin: Features

- 사용자가 설정한 시각에 알림을 줍니다
- 통화가 끝난 후, 메모를 작성하고 읽을 수 있습니다.
- 위젯을 통해 부모님과 통화한지 얼마나 지났는지 확인할 수 있습니다..
